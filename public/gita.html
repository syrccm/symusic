카포 어디에 끼우고 어떤 코드로 칠까?

플랫(♭) 코드가 많은 찬양곡을 쉬운 오픈 코드로 바꿔 연주할 수 있도록 도와드립니다. 원곡 키를 선택하면 카포 위치별 연주 코드를 한눈에 보여줍니다.

※ 원곡에 등장하는 코드들을 직접 입력하세요. 카포별로 어떻게 바뀌는지 아래 표에 표시됩니다.
② Eb 키의 카포 옵션
카드를 클릭하면 아래에 전체 코드 변환표가 표시됩니다
카포 3프렛 사용 시 코드 변환
원곡 코드 → 연주 코드
💡 사용 요령

카포의 원리: 카포를 N프렛에 끼우면 음이 N반음 높아집니다. 그래서 원곡 키보다 N반음 낮은 키의 쉬운 코드로 치면, 실제 나오는 소리는 원곡 키와 같아집니다.

예시: Eb 키의 곡에서 카포 3프렛을 끼우고 C키 코드(C, F, G, Am, Dm, Em)로 연주하면 실제로는 Eb 키로 울립니다. 찬양팀과 키를 맞출 수 있습니다.

🎸 쉬운 오픈 코드 키

가장 쉬움: G, C, D, A, E (바레 없이 연주 가능)

보통: Am, Em, Dm (마이너 오픈 코드)

어려움 (카포 권장): Eb, Ab, Bb, Db, F#, C# 등 플랫/샵 키는 바레 코드가 많아 초보자에게 부담스럽습니다.
