```
## 기기ID (로그인 없는 익명 식별)
src/utils/deviceId.ts:
- getDeviceId(): 브라우저에 저장된 기기ID 반환, 없으면 새로 생성 후 저장
- ID 형식: "device_" + 충분히 길고 랜덤한 문자열(crypto.randomUUID 기반, 추측 어렵게)
- 저장처: localStorage (키 예: 'symusic_device_id')

## 노트 타입 + Firestore CRUD
src/types/sermonNote.ts:
  interface SermonNote {
    id: string;
    worship: '주일'|'금철'|'QT'|'기타';   // STEP0에서 확인한 실제 명칭에 맞춤
    date: string;          // 'YYYY-MM-DD'
    passage?: string;      // 본문(선택)
    content: string;       // 내용(텍스트만)
    createdAt: number;
    updatedAt: number;
  }

src/utils/sermonNoteStore.ts (Firestore: sermonNotes/{deviceId}/notes/{noteId}):
- listNotes(): Promise<SermonNote[]>            // 최신순
- saveNote(note): Promise<void>                 // 신규/수정
- deleteNote(id): Promise<void>
- 모두 getDeviceId() 기준 경로 사용. 텍스트만 저장(첨부/이미지 없음).

작성 후 npx tsc -b 통과까지. 커밋은 마지막 STEP에서.
```

```
- 카드 3개 세로 배치:

## 1) 새 노트 작성

- [저장] → saveNote 후 목록으로 이동

## 2) 목록 보기
```

```
## 개별 노트 — [공유·백업] 누르면 액션 시트(하단에서 올라옴)
- 📤 텍스트로 공유/복사: navigator.share, 미지원 시 clipboard 복사
- ✉️ 이메일: mailto:(제목·본문 자동)
- 📥 이 노트 백업(JSON 파일): 그 노트 한 건을 JSON 파일로 다운로드
- 텍스트 형식: "[구분] 날짜 · 본문\n\n내용\n\n— Symusic 설교노트"

## 3) 관리 화면 (허브의 ⚙️ 관리)
- 📥 전체 백업(파일): 모든 노트를 JSON 파일로 다운로드
  (파일명 symusic-설교노트-YYYY-MM-DD.json, {app,version,exportedAt,deviceId,notes})
- 📤 전체 복원(파일): JSON 선택 → 합치기 / 덮어쓰기(추가 confirm) → saveNote로 반영
- 🔑 백업 코드로 복원: 기기ID 입력 → 그 코드의 Firestore 노트를 불러와 합치기
- 📤 전체 텍스트로 공유: 모든 노트를 텍스트로 한 번에
- 백업 코드 박스: 현재 기기ID 표시 + 복사 버튼 + "분실 시 복구 불가" 안내(gold 톤)
```

## STEP 4 — 노트 잠금 (화면 가림막, 4자리 PIN)
═══════════════════════════════════════════

```
설교노트에 잠금 기능을 추가해줘. "보안 암호화"가 아니라 "화면 가림막"이다.
(데이터는 평문 저장 유지, 잠금은 화면에서 가리는 용도)

## 데이터 추가
- SermonNote 에 locked: boolean 추가
- 설정 저장소(기기ID 기준): { defaultLock: boolean, pinHash: string|null }
  - PIN은 평문 저장 금지. 간단 해시(예: SHA-256)해서 pinHash로 저장.

## 관리 화면에 추가
- "PIN 설정/변경" (4자리 숫자 입력 → 해시 저장)
- "기본 잠금" 토글 (defaultLock): ON이면 새 노트가 기본 잠금
- 안내 한 줄: "중요한 노트는 이메일/파일로 백업해두세요"

## 새 노트 작성에 추가
- 🔒 잠금 토글 1개. 초기값 = defaultLock. 노트별로 켜고 끌 수 있음.
- locked=true로 저장하려면 PIN이 설정돼 있어야 함(없으면 PIN 먼저 설정 유도)

## 목록 보기
- locked 노트는 내용을 가리고 🔒 + "잠긴 노트" 정도만 표시
- 탭하면 PIN 입력 → 일치하면 그 노트 펼쳐 보기(세션 동안 유지)
- PIN 입력 화면에 "PIN 재설정" 버튼:
  → "잠금만 해제되고 노트는 삭제되지 않습니다" 안내 후 새 PIN 설정(또는 전체 잠금 해제)
- 자동 초기화/삭제 없음. 이메일 복구 없음.

npx tsc -b 통과까지.
```
